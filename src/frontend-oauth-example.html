Alloy Frontend OAuth Example

Step 1: Get JWT Token (Backend Required)

First, you need to get a JWT token from your backend. Your backend should call:

GET https://embedded.runalloy.com/users/{userId}/token
Headers:
  Authorization: Bearer {YOUR_API_KEY}
  x-api-version: 2025-09
  Accept: application/json

Note: Use GET method (not POST) and include the x-api-version header.
